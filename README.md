# 白い嘘 — Loops of Truth

> あなたは検事。容疑者は嘘をついている。
> **6 ターン以内に矛盾を突いて自白を引き出せ。**

選択肢ベースの本格推理パズルゲーム。失敗しても捜査メモは引き継がれ、
同じ事件を何度でも再挑戦できるハイブリッドループ式。
**Apple Intelligence 完全オフライン対応**で、あなたの尋問内容は
デバイスの外に出ない。

## ゲームの流れ

```
事件を選択
    ↓
容疑者のアリバイを聞く（弱点を探す）
    ↓
6 ターンの間、4 つの選択肢から質問を選ぶ
    ├─ 中立 ……… 安全だが効果は控えめ
    ├─ 追求 ……… 矛盾を引き出す軸
    ├─ 強気 ……… ハイリスク・ハイリターン
    └─ 証拠 ……… 集めた証拠で大ダメージ（要：証拠所持）
    ↓
自白ゲージ 100% で勝利、警戒度 100% / ターン切れで敗北
    ↓
[失敗] 同じ事件に再挑戦（証拠と捜査メモは保持）
[成功] 次の事件へ。10 事件で黒幕戦が解禁
```

## 4 つの質問カテゴリ

| カテゴリ | アイコン | 効果 | リスク |
|---------|---------|-----|------|
| **中立** | 💬 | 自白ゲージ ×0.6 | なし（警戒度わずかに低下） |
| **追求** | 🔍 | 自白ゲージ ×1.0 | 失敗で警戒度 +10% |
| **強気** | 🔥 | 自白ゲージ ×1.2 | 失敗で警戒度 +20% |
| **証拠** | 📄 | 自白ゲージ ×1.8 | 失敗で警戒度 +30%（要：証拠所持） |

---

## ハイブリッドループとは

このゲームの肝は **失敗が無駄にならない** 設計。

1. **1 ループ目**：手探りで質問する。容疑者のアリバイの弱点に触れると
   自動的に「証拠」が捜査メモに記録される
2. **失敗 → 再挑戦**：同じ事件で再開。証拠と捜査メモは保持
3. **2 ループ目以降**：選択肢に **「証拠を提示：◯◯◯」** が新たに登場
4. **証拠提示**：自白ゲージが一気に **+50% 以上** 上がる

→ 何度も挑戦するうちに真相が見えてくる、**Outer Wilds 系**の進化型推理体験。

## 動作環境

| 区分 | 推奨 | 最低 |
|------|------|------|
| iOS | 18.1 以上 | 17.0 |
| デバイス | iPhone 15 Pro 以降 | iPhone 12 以降 |
| Apple Intelligence | 対応 | 非対応でも遊べる |

非対応端末では **フォールバックモード**（決定論的な容疑者応答）で
全機能が動作。Apple Intelligence 対応端末ではより自然な対話と
動的事件生成（v2 以降）が体験できる。

## プライバシー

- ✅ **完全オフライン** — ネットワーク通信は一切行わない
- ✅ **オンデバイス AI** — Apple Foundation Models のみ使用
- ✅ **データ収集なし** — アナリティクスもクラッシュ報告も未収集
- ✅ **広告なし**
- ✅ **App 内課金なし**

すべてのプレイデータ（尋問ログ、捜査メモ、心理プロファイル）は
あなたの iPhone 内の SwiftData に保存され、外部に送信されることは
一切ありません。

## ヒント（初心者向け）

> 「どうやって勝つの？」と思ったら：

1. **1 ループ目は負けてOK**。証拠集めだと割り切る
2. アリバイに登場する単語（「妻」「夕食」「帰宅」「USB」など）を
   含む選択肢を選ぶ → 証拠が自動的に捜査メモに溜まる
3. **2 ループ目**：赤色の「**証拠を提示：◯◯◯**」を選ぶ
4. 証拠提示は 1 個で約 +50%。2 個提示すれば勝利確定
5. 強気な質問は終盤の決め手として温存

```bash
git clone https://github.com/changch223/claudebestgame.git
cd claudebestgame
open ClaudeBestGame.xcodeproj
# Xcode で iPhone 17 Pro Simulator を選択 → Cmd+R
```

### テスト

```bash
xcodebuild test \
  -scheme ClaudeBestGame \
  -destination 'platform=iOS Simulator,name=iPhone 17 Pro' \
  -only-testing:ClaudeBestGameTests
```

矛盾検出、証拠検出、選択肢生成、スコア計算）。
